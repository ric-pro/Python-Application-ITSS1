

####### Information to use load data from file  #########

test_data1 and test_data2 are the txt files which contain the data. Using 'Load data from file' button in the application, load any of these files. 

Then click on your desired option to see results.

If you click for "View Balance" button after loading the data. Balance amount will be shown after calculating the data given in the "test_data" files.
